## GraphQL with MongoDB

This code is a demo on using graphql to query data from the database (mongodb atlas)

### Current Features

* read data from the database

* add data from the database


### Packages used

* express

* mongoose

* graphql

* apollo-server-express

* @babel/cli

* @babel/core

* @babel/node

* @babel/preset-env

* dotenv

* nodemon
